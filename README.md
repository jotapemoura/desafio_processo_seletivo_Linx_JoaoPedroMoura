Aplicação Java Spring Boot para homologação de arquivos de vendas, desenvolvida para processar arquivos de vendas em formato posicional, persistir os dados em um banco de dados
relacional e disponibilizar uma interface web para visualização, filtragem e ordenação das transações.

Tecnologias Usadas: 

Java 21

Spring Boot 3.5.10 (Web, Data JPA, Thymeleaf)

H2 Database

Maven (Gerenciamento de dependências e Build)

JUnit 5 e Mockito (Testes Unitários e de Integração)

Bootstrap (Interface gráfica)


Como Rodar: 

Pré-requisito: Java 21.

Primeiro Passo: 

Clone este repositório para sua máquina, usando o comando no terminal: git clone https://github.com/jotapemoura/desafio_processo_seletivo_Linx_JoaoPedroMoura. Ou faça download do arquivo pelo repositório.

Segundo Passo:

Abra o arquivo no terminal e rode o comando: mvnw.cmd spring-boot:run ou ./mvnw spring-boot:run. Ou rode o executável ApiDesafioLinxApplication.java na IDE. Para visualizar página web, abra no navegador http://localhost:8080/ 

Testes:
Para rodar os testes abre o arquivo no terminal e rode o comando: ./mvnw clean test
